# Soma com uma quantidade arbitrária de suportes

Escreva a função `sum` que funcionaria assim:

`` `js
soma (1) (2) == 3; // 1 + 2
soma (1) (2) (3) == 6; // 1 + 2 + 3
soma (5) (- 1) (2) == 6
soma (6) (- 1) (- 2) (- 3) == 0
soma (0) (1) (2) (3) (4) (5) == 15
`` `

P.S. Dica: você pode precisar configurar o objeto personalizado para a conversão primitiva para sua função.
