# Reescreva 'se ..else' em '?'

Reescreva `if..else` usando vários operadores ternários` '?' `.

Para legibilidade, é recomendável dividir o código em várias linhas.

`` `js
deixar mensagem;

se (login == 'Empregado') {
mensagem = 'Olá';
} else if (login == 'Director') {
mensagem = 'Saudações';
} else if (login == '') {
mensagem = 'No login';
} outro {
mensagem = '';
}
`` `
