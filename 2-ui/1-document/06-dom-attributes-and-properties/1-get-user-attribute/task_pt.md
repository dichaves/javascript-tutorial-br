# Obter o atributo

Escreva o código para selecionar o elemento com o atributo `data-widget-name` do documento e para ler seu valor.

`` `html run
<! DOCTYPE HTML>
<html>
<corpo>

<div data-widget-name = "menu"> Escolha o gênero </ div>

<script>
/* seu código */
</ script>
</ body>
</ html>
`` `
